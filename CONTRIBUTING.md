# CONTRIBUTING - ikshana.ai

Please go through the document mentioned below to contribute to the library.

## Development Environment

<font size="4.5">Installing a development environment on Windows.</font>

* Clone the repository `git clone https://github.com/kaustubhharapanahalli/ikshana.git`
* Create a virtual environment `python -m venv ikshana`
* Install the required packages `python -m pip install -r requirements/all.txt`
